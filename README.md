# tic-tac-toe

A game of noughts and crosses (or tic-tac-toe) with a simple AI implementation ([minimax](https://en.wikipedia.org/wiki/Minimax)) 

## Compiling and Running
```
g++ game.cpp
```
Then:
* For Linux: 
```
./a.out
```
* For Windows:
```
./a
```

Note that row and column entries are zero indexed.

Enjoy!
